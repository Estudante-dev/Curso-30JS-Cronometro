# Cronômetro

Projeto de cronômetro digital do [#30ProjetosJavaScript](https://github.com/search?q=30ProjetosJavaScript), feito para ensinar HTML, CSS e JavaScript básicos.

## Funcionalidades

- Iniciar, pausar e reiniciar o tempo
- Exibição em horas, minutos, segundos e milissegundos (`00:00:00.000`)
- Status visual: Pronto, Em execução e Pausado
- Interface moderna com visual tecnológico

## Como abrir

Abra o arquivo `index.html` no navegador. Não é necessário instalar nada.
